## 一、产品概述

### 背景

热点新闻聚合市场竞争激烈（Google News、今日头条等），但大多数只是简单的信息搬运。HotPulse 的差异化在于：

1. **国内优先**：接入百度热搜 + 每天60秒，国内内容优先展示
2. **AI 独立模块**：AI 分析作为独立视图，用户可切换查看原始新闻或 AI 增值处理结果
3. **专业金融图表**：东方财富数据源 + ECharts K线图 + 成交量 + 板块排行
4. **全球视野**：同时覆盖国内 + 国际 3+ 个平台，一键切换

### 核心价值

- **节省时间：** AI 摘要让你 3 秒看懂一条热点
- **国内优先：** 60s 每日精选 + 百度热搜，第一时间了解国内大事
- **专业金融：** 11 个全球指数 + ECharts K线图 + MA均线 + 成交量 + 行业/概念板块排行
- **数据洞察：** 分类分布、热度排行、AI 分析报告，不只是看新闻更是理解趋势
- **心态建设：** 每日励志金句，正能量陪伴

### 目标用户

- 国内互联网用户为主（默认国内视图）
- 关注科技、财经、时事的互联网从业者
- 投资者（K线图 + 板块排行 + AI 市场分析）
- 信息密集型工作者（需要高效获取信息）

## 三、功能详情

### 3.1 新闻模块

**页面路由：** `/`

- **Hero 区域**：渐变横幅 + 问候语 + 日期 + 数据统计
- **视图切换**：全部新闻 / AI 分析
- **区域筛选**：国内 / 国际 / 全部
- **分类筛选**：科技 / 财经 / 娱乐 / 体育 / 健康 / 其他
- **AI 分析视图**：分类分布图 + 热度 Top 10
- **FeedCard**：排名徽章（金银铜）、平台专属颜色、热度进度条

### 3.2 股市模块

**页面路由：** `/stocks`

- **4 个 Tab**：行情数据 / K线详情 / 板块排行 / AI 分析
- **行情 Tab**：统计概览（监测数/上涨/下跌/平均涨跌）+ 多指数对比图 + 指数卡片（点击进入K线）
- **K线 Tab**：ECharts 蜡烛图 + MA5/10/20 均线 + 成交量柱状图 + 数据缩放
- **板块 Tab**：行业/概念板块切换 + 涨幅排行表格（涨跌家数/成交额）
- **AI Tab**：AI 市场分析报告
- **滚动行情条**：无缝循环滚动所有指数实时价格

### 3.3 分享功能

- 导航栏分享按钮 → 弹窗
- QR 码展示（qrcode.react）
- 复制链接（Clipboard API）
- 下载二维码图片（PNG）
- 分享到 Twitter / 微博

### 3.4 AI 分析功能

调用智谱 AI（glm-4-flash）：
- 每条热点自动生成中英文摘要 + 分类 + 情感分析
- cron 预生成 AI 市场分析报告
- asyncio.Semaphore(5) 并行处理

## 四、技术架构

### 技术栈

| 层级 | 技术 | 用途 |
|------|------|------|
| 前端框架 | Next.js 16 (App Router) | SSR + 静态生成 |
| 前端语言 | TypeScript | 类型安全 |
| UI 样式 | Tailwind CSS v4 | 响应式设计（渐变/毛玻璃/动画）|
| 图表库 | ECharts (echarts-for-react) | K线图 + 成交量 |
| 对比图 | recharts | 多指数走势对比 |
| QR码 | qrcode.react | 分享二维码 |
| 后端框架 | Python FastAPI | RESTful API |
| AI 服务 | 智谱 AI (glm-4-flash) | 摘要、分类、情感、市场分析 |
| 数据库 | Supabase (PostgreSQL) | 主数据存储 |
| 股市数据 | 东方财富 HTTP API | 实时行情 + K线 + 板块排行 |
| 定时触发 | cron-job.org (免费) | 外部 POST 触发 |
| 部署 | Vercel (前端) + Railway (后端) | 生产环境 |

### 数据库表

| 表名 | 说明 |
|------|------|
| trending_items | 热点新闻条目（AI摘要/分类/情感） |
| quotes | 励志名言（500+） |
| stock_indices | 股市指数最新快照（OHLCV） |
| stock_history | K线历史数据（OHLCV） |
| market_analysis | AI 市场分析报告 |

### API 端点

| 端点 | 说明 |
|------|------|
| `GET /api/trending` | 新闻列表（category/region/page） |
| `GET /api/trending/stats` | 统计（分类分布 + 热度Top10） |
| `GET /api/stocks` | 指数快照 |
| `GET /api/stocks/kline/{symbol}` | K线 OHLCV 数据 |
| `GET /api/stocks/sectors` | 板块排行（industry/concept） |
| `GET /api/stocks/history/{symbol}` | 历史数据 |
| `GET /api/stocks/analysis` | AI 市场分析 |
| `POST /api/cron/fetch-trending` | 新闻抓取（cron） |
| `POST /api/cron/fetch-stocks` | 股市抓取（cron） |
